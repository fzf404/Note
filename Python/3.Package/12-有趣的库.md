# 有趣的py库

## 模拟按键发送

```python
import pyautogui, time

# 防止受当前键盘影响
time.sleep(0.3)
# 发送单个按键
pyautogui.press('f')
pyautogui.press('enter') # shift ctrl
# 发送多个按键
pyautogui.typewrite('dir')
pyautogui.press('enter')
# 组合按键
pyautogui.keyDown('ctrl')
pyautogui.press('a')
pyautogui.keyUp('ctrl')
pyautogui.press('del')
```
