## Folium

```python
import folium

# 初始化地图，坐标为沈阳市
m = folium.Map(
    location=[41.7980,123.4395],
    zoom_start=14
)

# 悬浮弹出信息
popup = '嘿！我是火车站！'

# 添加marker到地图 (坐标信息，点击弹出内容，鼠标悬浮弹出内容)
folium.Marker([41.7917,123.3889], popup=popup, tooltip='沈阳站').add_to(m)
folium.Marker([41.8166,123.4297], popup=popup, tooltip='沈阳北站').add_to(m)
```
