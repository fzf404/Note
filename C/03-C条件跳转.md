# 条件跳转指令

## while

```c
while(scanf("%ld", &num) == 1);		// 跳过第一个数字
while(scanf("%*", &*) == 1)			// 获取连续输入
```

### 运算符号

- 条件运算：`?:`

  ```c
  x = (y<0)?y:-y;		// c中唯一一个三元运算符
  ```

## 跳出

```c
	while(scanf("%d", &i))
	{
		if (i > 100)
		{
			printf("its too big.\n");
			continue;		// 回到while获取输入值
		}
		printf("its a right nember");
		break;				// 跳出循环
	}
```

## `switch()`

```c
switch(x)
{
    case 1:
        printf("1");
        break;
    case 2:
        printf("2");
        break;
    default:
        printf("404")
}
```
