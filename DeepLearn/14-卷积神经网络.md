> 使用卷积核提取图片特征值

### 二维卷积

```python
from torch.nn import functional as F

# 图像信息:个数、通道数、长宽像素数
x = torch.randn(1,3,28,28)
# 核数、通道数、核的长宽
w = torch.rand(16,3,5,5)
b = torch.rand(16)
# 核的步长值，图像向外补充像素数
out = F.conv2d(x,w,b,stride = 1,padding = 1)

import torch.nn as nn
# 图像个数、通道数、核大小、步长、补偿
layer = nn.Conv2d(1,3,kernel_size = 3,stride = 1,padding = 1)
out = layer.forward(x)
```
