## npm

```bash
npm init -y
npm install react --save
```

## yarn

```bash
yarn init
yarn add react --save	# 添加新插件
yarn install		# 拉取旧插件
```

## 脚手架

```bash
yarn add -g create-react-app
create-react-app my-app
# 预览
yarn start
# 暴露配置文件
yarn eject
```
