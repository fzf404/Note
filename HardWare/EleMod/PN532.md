# PN532加密卡复制

## 种类

| 名称  | 类型     | 区别                           |
| ----- | -------- | ------------------------------ |
| M1    | IC卡     | 0扇区不可修改，其他可反复擦写  |
| UID   | IC复制卡 | 所有扇区可反复擦写             |
| CUID  | IC复制卡 | 屏蔽UID的可使用CUID            |
| FUID  | IC复制卡 | 屏蔽CUID的可使用FUID           |
| UFUID | 合成卡   | UID与FUID合成，封卡后成为UID卡 |
