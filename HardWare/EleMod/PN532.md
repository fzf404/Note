# PN532 加密卡复制

## 种类

| 名称  | 类型      | 区别                                |
| ----- | --------- | ----------------------------------- |
| M1    | IC 卡     | 0 扇区不可修改，其他可反复擦写      |
| UID   | IC 复制卡 | 所有扇区可反复擦写                  |
| CUID  | IC 复制卡 | 屏蔽 UID 的可使用 CUID              |
| FUID  | IC 复制卡 | 屏蔽 CUID 的可使用 FUID             |
| UFUID | 合成卡    | UID 与 FUID 合成，封卡后成为 UID 卡 |
