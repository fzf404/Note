# Dos常用符号

## @    命令行回显

- 屏蔽当前行的回显

  ```powershell
  @echo off
  ```

## \>    重定向符

> \>    传递结果至后面
> 
> \>\>    传递结果但不覆盖

## |      管道命令符

> |    将第一条命令的结果当作第二条命令的参数使用
> 
> ||    第一条命令失败后执行下一条：

  ```powershell
  di || echo Enter_Error
  ```

## &    组合命令符

> &			从前向后执行
>
> &&		命令失败后不向后执行
> 

  ```powershell
dir test && (rd /s/q test & echo rd_run)	# 存在test则删除
  ```

## ,;    逗号与分号

- ，    可当空格使用
- ；    当命令相同时隔离不同目标

## ()    代码块

- （）中的代码被看作一个整体

## %1    参数

- 接收执行时输入的参数
