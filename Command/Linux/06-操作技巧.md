## 快捷键

| 快捷键     | 用途                                       |
| ---------- | ------------------------------------------ |
| Ctrl+AEBFL | 开头、结尾、前、后、清屏                   |
| Alt+BF     | 前一个单词、后…                            |
| Ctrl+DTU   | 删除光标所在位置的字符、调换位置、剪贴该行 |
| Alt+DTUp   | 剪贴到词尾、单词调换位置、转大写           |
| Ctrl+Y     | 粘贴                                       |
| Alt+?      | 显示所有可能的自动补全                     |
| Ctrl+R     | 反向增量搜索命令                           |
| !!         | 上次执行的命令                             |
