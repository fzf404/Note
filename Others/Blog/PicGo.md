# PicGo

> 下载：[Releases](https://github.com/Molunerfinn/picgo/releases)

## Gitee

1. 安装`node-js`

2. 安装插件`Gitee`

3. 只需填写`repo`及`token`

   ![image-20200718213813253](https://img-1257284600.cos.ap-beijing.myqcloud.com/2020/20200718213813.png)

## Commond

```bash
npm i picgo -g
git install gitee
# 配置文件
~/.picgo/config.json
picgo set uploader
```
