# PicGo

> 下载：[Releases](https://github.com/Molunerfinn/picgo/releases)

## Gitee

1. 安装`node-js`

2. 安装插件`Gitee`

3. 只需填写`repo`及`token`

   ![image-20200718213813253](https://gitee.com/nmdfzf404/Image-hosting/raw/master/20200718213813.png)

## Commond

```bash
npm i picgo -g
git install gitee
# 配置文件
~/.picgo/config.json
picgo set uploader
```
