> 网络

## 常用命令

### ping

> 发送 `ICMP ECHO_REQUEST`到主机

### tracepath

> `tracerote`
>
> 显示本地到服务器的跳数

### ftp

| 命令         | 含义             |
| ------------ | ---------------- |
| ftp server   | 连接到ftp服务器  |
| get filename | 从ftp获取文件    |
| lcd ~        | 切换本机接收目录 |

### wget

> 下载网络上的东西

### ssh

> 服务器重装后，key会改变
>
> 要想再次连接
>
> 需要删除`~/.ssh/know_hosts`

### scp

> `scp usr@server:filename yourfile`
