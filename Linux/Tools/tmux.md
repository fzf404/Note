> 终端复用神器

## 入门

- Session

  >  一个tmux会话

- Window

  > 一个窗口

- Pane

  > 一个区域

## 快捷键

> `ctrl+b`: Prefix键

| 快捷键   | 功能        |
| -------- | ----------- |
| %        | 左右分屏    |
| “        | 上下分屏    |
| o/;      | 切换Pane    |
| z        | 全屏Pane    |
| x/Ctrl+D | 关闭Pane    |
| c        | 新建Window  |
| &        | 关闭Window  |
| :        | 新建Session |
| s/w      | 切换Session |
| d        | 离开Session |
| tmux a   | 进入Session |
