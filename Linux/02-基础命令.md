## 权

## 限

```bash
id # 显示用户身份

chmod <path> # 管理权限
    -R # 递归增加
    755 # 4+2+1 读|写|运行
    +x # 所有用户增加运行权限
    o-x # 其他用户减去运行权限
    # u g o a 当前用户|当前组|其他用户|全部用户

chmod user:group # 更改文件所有者及组别
chgrp user:group # 更改文件所有组
    -R # 递归修改
```

## 用户管理

```bash
who # 显示已登录用户

su <user_name> # 切换用户

passwd # 修改密码
sudo passwd # 修改管理员密码
```

## 操作符

```bash
# 重定向
> # 覆盖
>> # 追加

# 正确信息重定向
> success.log
# 错误信息重定向
2> error.log
# 全部重定向到同一个文件
>all.log 2>&1

# 例子
echo "Some Text" > hello.txt
cat hello.txt
echo "Some Text2" > hello.txt
cat hello.txt
echo "Some Text3" >> hello.txt
cat hello.txt

# 管道
|

# 例子
ls -l / | grep bin # 在根目录搜索bin
```



## 通配符

```bash
`*`：匹配任意多个字符

`?`：匹配任意单个字符

`[char]`：匹配任意字符集中的字符

# 例子

`[[:!digit:]]*`：不已数字开头的文件

`*[[:lower:]123]`：已小写字母或123结尾的文件
```

## 运行

```bash
# 后台运行
<commond> & # 当前终端后台执行
# 后台任务
jobs -l
# 取出后台任务
fg $1
# 任务放置后台
Ctrl + Z
# 系统后台运行
nohup <command> & 

# 停止
kill <sigle> <uid>
| ---- | ---- | --------------------------------- |
| 1    | HUP  | 挂起                              |
| 2    | INT  | Ctrl-c                            |
| 9    | KILL | 内核立即终止它                    |
| 15   | TERM | 默认信号,程序仍然运行接收到会停止 |
| 18   | CONT | 继续运行                          |
| 19   | STOP | 停止运行                          |
```
