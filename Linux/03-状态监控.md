## 

## 型号

```bash
# cpu信息
cat /proc/cpuinfo
# 发行版
cat /proc/version
```

## 占用

```bash
top # 动态展示进程信息
htop # 动态展示进程信息-更好看

free -h # 内存占用
df -h # 存储空间占用
du -h # 文件详细占用
du -sh * # 当前目录占用
```

## 进程

```bash
ps x # 全部进程
  PID # 进程ID
  TT # 进程的控制终
  STAT # 状态
    R-S-D-T-Z # 运行|睡眠|等待IO睡眠|停止|终止

ps -ef # 标准格式
  UID		# 用户ID
  PID		# 进程ID 
  PPID	# 父进程ID 
  C			# CPU占用率
  STIME	# 进程启动时间 
  TTY		# 运行进程的终端 ? pts/0 非终端|网络连接终端
  TIME  # 进程占用CPU时间 
  CMD   # 命令的名称和参数

ps aux # 更多信息
  USER	# 用户名 
  %CPU	# CPU占用率
  %MEM	# 内存占用率 
  VSZ		# 虚拟內存量（KB） 
  RSS		# 固定內存量（KB）
  STAT	# 进程状态 
  START	# 进程启动时间 
  TIME	# 进程占用CPU时间
  CMD   # 命令的名称和参数

# 例子
ps -ef | grep <>
```
