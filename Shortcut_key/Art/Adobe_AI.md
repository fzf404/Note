# illustrator快捷键

### 绘制基础

| 快捷键    | 功能             |
| --------- | ---------------- |
| [space]   | 抓手             |
| Ctrl+Y    | 显示所有元素     |
| V/A       | 选择工具         |
| M         | 矩形工具         |
| E         | 自由变换         |
| Alt/Shift | 圆心为轴/正体    |
| X         | 切换到描边色     |
| Shift+X   | 互换颜色         |
| /,.       | 无颜色/颜色/渐变 |

### 绘制案例

| 快捷键        | 功能      |
| ------------- | --------- |
| Ctrl+G        | 编组      |
| Ctrl+][       | 前移/后移 |
| Shift+Ctrl+][ | 顶层/底层 |
| Ctrl+D        | 再次变换  |

### 路径查找器

| 快捷键        | 功能        |
| ------------- | ----------- |
| Ctrl+Shift+F9 | 路径查找器  |
| I             | 吸管        |
| Ctrl+F/B      | 粘贴在前/后 |

### 线性工具

| 快捷键 | 功能      |
| ------ | --------- |
| \      | 直线工具  |
| R/O    | 旋转/镜像 |
| Ctrl+D | 再次执行  |
| Ctrl+J | 连接      |

### 形状生成器

| 快捷键  | 功能       |
| ------- | ---------- |
| Shift+M | 形状生成器 |

### 曲线
