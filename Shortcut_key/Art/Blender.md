# Blender

## 操作

| 快捷键        | 功能                |
| ------------- | ------------------- |
| g/s/r         | 移动旋转缩放        |
| Alt+[g/s/r]   | 重置 gsr            |
| [x/y/z]       | 在 x/y/z 轴进行缩放 |
| Shift+[x/y/z] | 反选轴向            |
| i             | 打关键帧            |
