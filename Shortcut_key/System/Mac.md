## 系统

| 快捷键                        | 作用           |
| ----------------------------- | -------------- |
| Command + Control + F         | 全屏           |
| Command + Shift + H           | 访达查看家目录 |
| Command + Control + Shift + 4 | 截图存至剪贴板 |
| Command + Alt + Esc           | 任务管理器     |
| Command + Control + Space     | 输入 Emoji     |
| Command + Shift + F           | 应用内全屏     |
