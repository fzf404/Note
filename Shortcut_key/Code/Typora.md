# Typora\_快捷键

## 快速工作流

### 选择

| 快捷键       | 功能            |
| ------------ | --------------- |
| Ctrl+L       | 选中当前行      |
| Ctrl+D       | 选中当前词      |
| Ctrl+Shift+D | 删除当前词      |
| Ctrl+Shift+V | 粘贴纯文本      |
| Ctrl+Shift+C | 复制为 MarkDown |

### 编辑

| 快捷键        | 功能       |
| ------------- | :--------- |
| Ctrl+P        | 快速打开   |
| Ctrl+H        | 替换       |
| F3/Enter      | 查找下一个 |
| Ctrl+Shift+[] | 列表       |

### 功能

| 快捷键       | 功能       |
| ------------ | ---------- |
| Ctrl+Shift+L | 侧边栏     |
| Ctrl+/       | 源代码格式 |
| Ctrl+Enter   | 增加一行   |
| Ctrl+Shift+K | 插入代码块 |
| Ctrl+K       | 链接       |
| Ctrl+Shift+I | 插入图片   |
